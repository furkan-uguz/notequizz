### NoteQuizz

NoteQuizz is a browser-based music challenge game that you should choose true notes on treble staff with given time. Also you can do praticeses without racing time. This app will help you for reading the music's better and faster. You dont need any installiation or any program for uses. 

- Simple, responsive and stylish design,
- Easy to use,
- Suitable for all platforms(Mobile, Tablet, PC etc.) and browsers,
- Understandable animations, icon sets, colors
